# colorful.css  

## Introduction  

A bootstrap-like CSS lib，but lighter and simplier.
Size: 10kb (README.md and sample.html concluded)

## Usage

See more on ./sample.html
Some classes can be seen below ↓

### Buttons

#### Size unit/px

* .btnLarge 10x12
* .btnMedium 6x12
* .btnSmall 1x5

#### Buttons-Colours

* .buttonGreen
* .buttonWhite
* .buttonPurple
* .buttonBlue
* .buttonOrange
* .buttonRed

### Alerts (Size can be set)

#### Alerts-Colours

* .alertGreen
* .alertYellow
* .alertBlue
* .alertRed

### Process Bars (Size can be set)

* .pb (Purple)
* .pbStiped (Purple Striped)
* .pbGreen
* .pbYellow
* .pbRed

### Panels (Size can be set)

* .panelDefault (Grey)
* .panelGreen
* .panelYellow
* .panelPurple
* .panelBlue
* .panelRed

### Labels (Size can't be set)

* .labelDefault (Grey)
* .labelGreen
* .labelYellow
* .labelPurple
* .labelBlue
* .labelRed

### Thumbnails (Size: 自己写)

Default： 200x200px
